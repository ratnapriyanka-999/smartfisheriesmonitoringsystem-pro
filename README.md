Smart Fisheries Monitoring System

📌 Overview

This project is an IoT-based smart monitoring system designed to track water quality parameters such as oxygen levels in real time. It helps fish farmers maintain optimal conditions and prevent losses.

🚀 Features

- Real-time monitoring dashboard
- Oxygen level tracking and alert system
- User-friendly web interface
- Data visualization for better insights

🛠️ Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Python
- Tools: IoT sensors integration
